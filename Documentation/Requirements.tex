\section*{1.4 Scope, Assumptions \& Constraints}
\begin{itemize}
\item \textbf{In-Scope:} CSV loading, C HTTP server, HTML/JS/Chart.js, save-graph.
\item \textbf{Out-of-Scope:} Real-time scraping, authentication, payments.
\item \textbf{Assumptions:} Internet, accurate CSV, single-user testing.
\item \textbf{Constraints:} Free tools, C + optional Python, 2–3 week prototype.
\item \textbf{Testing Devices:} Laptop (Ubuntu/Windows) + Android smartphone.
\item \textbf{Success Criteria:} Demo: select → type → graph → save PNG.
\end{itemize}
